# Práctica de Python

Repositorio con ejercicios de Python resueltos como práctica personal,
ordenados por nivel de dificultad.

## Objetivo

Reforzar lógica de programación y fundamentos de Python, incluyendo:
- Bucles (`while`, `for`)
- Listas
- Funciones
- Control de flujo

## Estructura del repositorio

- `01_basico`
- `02_while_for`
- `03_listas`
- `04_funciones`

## Tecnologías

- Python 3.x

## Nota

Ejercicios realizados con fines educativos y de aprendizaje continuo.
